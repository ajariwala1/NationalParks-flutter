# National Parks App

This project is the completed version of the National Parks app for Module 4 of the Mobile Application Development course.

The project includes:

* Swipeable introduction screens
* Randomized onboarding images and captions
* Provider-based shared state management
* Local data persistence using SharedPreferences
* Integration with the National Park Service API
* JSON decoding using Dart models
* State selection and park search functionality
* Dynamic data loading using HTTP requests
* Grid-based park browsing
* Detailed park information screens
* Park activities, fees, directions, and operating hours
* Navigation between multiple screens

Before starting, run:

```bash
flutter pub get
```

to download any required dependencies.

⚠️ IMPORTANT: The app won't work as-is. You need to register and get your own API key from the National Park Service API to use the app.

Once the project is ready, open `main.dart` and run the application on a simulator, emulator, or physical device.
